■ 소개
-----------------------------------------------------------
Moonstone Island의 비공식 한국어 번역 패치입니다.
게임의 UI, 대화, 퀘스트, 아이템, 카드 등 주요 텍스트를
한국어로 즐길 수 있습니다.

호환 게임 버전: 1.7.2365 (Steam)


■ 번역 범위
-----------------------------------------------------------
- 시스템 UI (1,407개 문자열)
- NPC 대화 (11명, 3,053개 대사)
  : Ferra, Sterling, Ofelia, Paolo, Tobin, Gaiana,
    Cleo, Zed, Rowan, Waldo, Ossono
- 퀘스트 (96개)
- 카드 설명 (53개)
- 아이템 설명 (85개)
- 우편 (40개)

총 5,460개 이상의 문자열이 번역되었습니다.


■ 설치 방법
-----------------------------------------------------------
1. 이 폴더 전체를 원하는 위치에 압축 해제합니다.

2. "한글패치.exe"를 실행합니다.
   (Windows가 차단하는 경우 "추가 정보" > "실행" 클릭)

3. 게임 경로가 자동으로 감지됩니다.
   감지되지 않으면 직접 입력합니다.
   (Steam > Moonstone Island 우클릭 > 관리 > 로컬 파일 탐색)

4. 메뉴에서 [1] 한글 패치 설치를 선택합니다.

5. 설치가 완료되면 게임을 실행합니다.

6. 게임 설정에서 언어를 "한국어"로 선택합니다.
   (내부적으로 러시아어 슬롯을 사용하지만, 옵션에는 한국어로 표시됩니다)

7. 설정 적용 후 게임을 재시작해 한글 적용 여부를 확인합니다.

※ "관리자 권한" 오류가 나오면:
   한글패치.exe 우클릭 > "관리자 권한으로 실행"


■ 패치 제거 (원본 복구)
-----------------------------------------------------------
1. "한글패치.exe"를 실행합니다.
2. 메뉴에서 [2] 패치 제거를 선택합니다.
3. 원본 package.nw가 자동으로 복구됩니다.

※ 패치 설치 시 원본 파일이 자동으로 백업됩니다.
   (package.nw.backup)

※ 백업을 분실한 경우:
   Steam > Moonstone Island 우클릭 > 속성
   > 설치된 파일 > 게임 파일 무결성 확인


■ 게임 업데이트 시
-----------------------------------------------------------
Steam이 게임을 업데이트하면 package.nw가 원본으로
교체됩니다. 이 경우 패치를 다시 설치하면 됩니다.

※ 새 게임 버전에서 패치가 호환되지 않을 수 있습니다.
   문제 발생 시 패치 업데이트 버전을 확인해 주세요.


■ 현재 품질 상태 (2026-02-14 기준)
-----------------------------------------------------------

■ 번역 용어
-----------------------------------------------------------
- Stamina = 스태미나 (HP = 체력)
- Tavern = 선술집
- Selling Bin = 판매함
- Hand (카드) = 손패
- Recipe (연금술) = 도안


■ NPC 말투 안내
-----------------------------------------------------------
각 NPC는 고유한 말투를 가지고 있습니다:
- Paolo (목수): 스페인어 섞인 다정한 말투
- Tobin (어부): 느긋하고 친근한 사투리
- Gaiana (약초사): 극도로 간결하고 직설적
- Ossono (선술집 주인): 시끌벅적하고 호탕한 말투
- 기타 NPC: 각 성격에 맞는 해요체 변형
